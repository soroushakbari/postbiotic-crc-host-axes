# Data sources

This repository accompanies a transcriptomic analysis of postbiotic-related host axes in colorectal cancer.

Raw public data are not redistributed here.

The analyses use:

- TCGA colorectal cancer cohorts, accessed through the Genomic Data Commons and TCGAbiolinks.
- GSE39582, accessed through the Gene Expression Omnibus.

Processed non-identifiable summary outputs, curated gene-set definitions, tumour microenvironment marker signatures, final tables, and final manuscript figures are provided for reproducibility.

Users should download public raw data from the original repositories when reproducing the full workflow.
