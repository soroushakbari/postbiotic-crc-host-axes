# Reproducibility notes

The repository is intended as a clean reproducibility package accompanying the manuscript.

It includes:

- Core R scripts used for postbiotic-axis scoring and downstream analyses.
- Curated postbiotic-axis definitions.
- Marker-based tumour microenvironment signature definitions.
- Final processed summary tables.
- Final manuscript figures.

Exploratory scripts, draft files, raw literature-screening exports, local logs, and manuscript documents are not included because they are not required to reproduce or inspect the reported analyses.
